# MCP CLI Source Directory

이 디렉토리는 MCP (Model Context Protocol) CLI의 핵심 실행 로직과 노드 기반(Typescript) 소스 코드를 보관합니다.

---

## 📂 파일 및 디렉토리 구성

| 파일/디렉토리명 | 목적 및 주요 역할 |
|-----------------|-------------------|
| `cli.ts` / `index.ts` | CLI 애플리케이션의 진입점(Entry Point). 환경 설정 로드 및 전역 명령어 파싱을 담당 |
| `commands/` | 사용자가 터미널에서 입력할 수 있는 개별 명령어(예: setup, export 등)의 비즈니스 로직 처리 |
| `utils/` | 파일 I/O, 환경 변수 파싱, 경로 조작 등 CLI 동작에 필요한 공통 유틸리티 함수 모음 |

---

## 💡 주요 특징
- `.ai/` 하위 시스템이 프롬프트/마크다운 기반의 "인스트럭션" 이라면, `src/` 폴더는 이를 **구동하고 관리하기 위한 애플리케이션 코어(TypeScript)** 영역입니다.
- 명령어 추가나 CLI 도구의 구조 변경 시 이곳의 로직을 수정 후, 프로젝트 루트에서 빌드(npm build)를 수행하여 반영합니다.
