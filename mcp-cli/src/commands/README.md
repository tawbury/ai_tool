# MCP CLI Commands Directory

이 디렉토리는 MCP CLI 애플리케이션의 터미널 명령어(Subcommands)에 대한 비즈니스 로직을 보관합니다.

---

## 📂 주요 파일 구성

| 파일명 | 명령어 역할 및 기능 |
|--------|---------------------|
| `init.ts` | 시스템 초기화 및 기본 환경 변수/파일 셋업을 담당하는 `init` 명령어 로직 |
| `install.ts` | 어댑터와 매핑 룰을 기반으로 `.claude`, `.cursor` 등 각 AI 클라이언트 폴더에 설정을 배포하는 `install` 명령어 로직 |
| `sync.ts` | 중앙 룰(`.ai/rules`)과 클라이언트 설정 간의 변경사항을 양방향/단방향으로 동기화하는 `sync` 명령어 로직 |
