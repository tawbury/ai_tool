# Templates Directory

This directory contains reusable document templates and activation contract examples.

## Activation Templates

| File | Purpose |
|---|---|
| `activation.template.yaml` | Activation contract v0 example. |
| `activation.v1.template.yaml` | Activation contract v1 example with runtime mode, rule references, and loader overrides. |

## Base Templates

`_base/` contains shared template fragments for metadata, schema, mapping, planning, execution, management, and rule structure.

## Boundary

Templates are source material for humans and AI agents. The current runtime validates and reads selected templates, but it does not auto-fix, materialize files, execute workflows, dispatch workers, or generate adapters.
