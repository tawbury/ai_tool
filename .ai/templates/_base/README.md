# Base Templates

This directory contains reusable template fragments used by higher-level document templates.

## Files

| File | Purpose |
|---|---|
| `base_template.md` | Common document template base. |
| `base_meta.md` | Shared metadata fields. |
| `base_schema.md` | Shared schema guidance. |
| `base_mapping.md` | Mapping guidance. |
| `base_rules.md` | Base rule structure. |
| `planning_base.md` | Planning-stage base. |
| `execution_base.md` | Execution-stage base. |
| `management_base.md` | Management-stage base. |

These files are template source material, not runtime execution instructions.
