# HR Skills Directory

이 디렉토리는 Human Resources(HR) 에이전트가 구성원(AI 에이전트 포함) 역량 평가, 과업 분배, 퍼포먼스 리뷰를 위해 사용하는 스킬(`.skill.md`)들을 보관합니다.

---

## 📂 주요 파일 구성

| 파일명 | 스킬 역할 및 기능 |
|--------|-------------------|
| `hr_onboarding.skill.md` | 신규 역할/태스크(Task) 생성 시 검증 및 온보딩 파이프라인 수행 |
| `hr_level_check.skill.md` | 태스크 난이도 및 수행 결과에 따른 L1(Junior) / L2(Senior) 레벨 판별 로직 |
| `hr_role_management.skill.md` | 프로젝트 내 에이전트들의 권한, 책임 및 R&R(Role & Responsibility) 조정 |
| `hr_task_distribution.skill.md` | 에이전트 특성에 맞춘 적절한 과업(Task) 분배 및 할당 |
| `hr_performance_lifecycle_unified.skill.md` | 태스크 수행 시작부터 종료까지의 퍼포먼스 추적 및 관리 통합 스킬 |
| `hr_report_emit.skill.md` | 최종 인사(HR) 평가 보고서 자동 산출 및 저장 |
| `hr_goal_feedback_unified.skill.md` / `hr_capacity_development.skill.md` | 목표 달성도 피드백 및 에이전트 역량 강화 가이드라인 제안 |
| `hr_organization_design.skill.md` | 시스템 또는 팀(조직) 구조의 최적화 설계 |
| `hr_analytics_unified.skill.md` / `turnover_prediction.skill.md` | HR 데이터 시계열 분석 및 가동률, 과부하/이탈률 예측 모델링 |
| `hr_career_management_unified.skill.md` / `hr_development_programs_unified.skill.md` | 중장기 역량(커리어) 발전 프로그램 기획 및 적용 |
| `hr_context_intelligence_unified.skill.md` | 평가 컨텍스트 및 히스토리 기반의 인텔리전스 분석 |
| `hr_template_management.skill.md` | HR 양식(Report, Task) 템플릿의 검수 및 배포 |
