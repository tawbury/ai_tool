# Contents Creator Frameworks (Shared)

이 디렉토리는 콘텐츠 기획과 제작 전반에 걸친 핵심 프레임워크 스킬들을 보관합니다.
단일 콘텐츠가 아닌 전체 플랫폼의 전략을 수립할 때 여러 에이전트가 참조할 수 있는 공통 지침 역할을 합니다.

---

## 📂 파일 구성

| 파일명 | 스킬 역할 및 목적 |
|--------|-------------------|
| `contents_strategy_framework.skill.md` | 장기적인 콘텐츠 방향성 및 브랜딩 전략 프레임워크 |
| `contents_creation_framework.skill.md` | 콘텐츠 생성부터 발행까지의 파이프라인 프레임워크 |
| `media_integration_framework.skill.md` | 텍스트, 비디오, 시각 애셋 등 멀티미디어 통합 프레임워크 |
| `quality_assurance_framework.skill.md` | 콘텐츠 퀄리티 검수 및 SEO, 정책 준수 여부 점검(QA) 프레임워크 |
