# Shared AI CLI Skills

This directory contains reusable AI CLI tooling skills.

## Subdirectories

| Path | Purpose |
|---|---|
| `hook-creator/` | Create and configure lifecycle hooks. |
| `skill-creator/` | Create or update reusable AI skills. |
| `slash-command-creator/` | Create shared command wrappers or slash commands. |
| `subagent-creator/` | Create specialized AI agent definitions. |
| `youtube-collector/` | Collect YouTube channel/video metadata and transcripts. |

## Boundary

These skills are shared assets. Tool-specific behavior may be documented inside a skill when necessary, but reusable behavior should stay in `.ai/skills/_shared/ai_cli/`.
