# Contents Creator: Text Skills

이 디렉토리는 텍스트 중심의 콘텐츠(기사, 블로그, 이북 등) 생성 및 품질 고도화를 담당하는 스킬들을 보관합니다.

---

## 📂 파일 구성

| 파일명 | 스킬 역할 및 목적 |
|--------|-------------------|
| `contents_writing_fundamentals.skill.md` | 기본 작문 톤앤매너 설정 및 윤문 가이드 |
| `contents_optimization.skill.md` / `contents_analytics.skill.md` | 검색 엔진 최적화(SEO) 및 콘텐츠 데이터 분석 스킬 |
| `ebook_structuring.skill.md` / `ebook_writing.skill.md` | 목차 구성부터 챕터별 내용 작성까지 전자책 집필 스킬 |
| `ebook_editing.skill.md` | 초고 교정교열(Editing) 및 퇴고 스킬 |
| `visual_contents_strategy.skill.md` / `contents_integration.skill.md` | 텍스트와 시각 자료(이미지/영상)의 배치 기획 및 통합 가이드 |
| `audience_analytics.skill.md` / `ebook_platform_strategy.skill.md` | 독자 데이터 분석 및 타겟 플랫폼(리디, 크몽 등) 맞춤형 최적화 |
