# Contents Creator Skills Directory

이 디렉토리는 Contents Creator 에이전트가 사용하는 콘텐츠 기획, 텍스트/시각자료 생성, 품질 검수 목적의 스킬들을 보관합니다.

---

## 📂 파일 및 디렉토리 구성

콘텐츠 포맷 및 목적에 따라 다음과 같이 하위 디렉토리 단위로 스킬이 구분됩니다:

| 분류명 (디렉토리) | 목적 및 주요 기능 |
|-------------------|-------------------|
| `business/` | 비즈니스 문서, 제안서, 프레젠테이션 스크립트 작성 및 톤앤매너 교정 스킬 |
| `text/` | 블로그, SEO 아티클, 홍보 문구 등 텍스트 기반 콘텐츠 생성 및 최적화 스킬 |
| `visual/` | UI 디자인 가이드, 이미지 프롬프트 생성, 브랜드 아이덴티티 시각화 조율 스킬 |
| `video/` | 비디오 스크립트 기획, 스토리보드 작성, 시각적 트랜지션 및 대본 기획 스킬 |
| `interactive/` | 사용자와 상호작용하는 콘텐츠(예: 인터랙티브 웹 기획, 마이크로카피) 관련 스킬 |

이러한 스킬들은 에이전트가 각 매체에 맞는 퀄리티 보증과 일관된 브랜드 메시지를 유지하는 데 활용됩니다.
