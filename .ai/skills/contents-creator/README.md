# Contents Creator Skills

This directory contains content creation skills grouped by format and business use case.

## Subdirectories

| Path | Purpose |
|---|---|
| `business/` | eBook market, audience, marketing, and monetization skills. |
| `interactive/` | Interactive design and UX/UI skills. |
| `text/` | Writing, editing, optimization, analytics, and eBook structure skills. |
| `video/` | Storyboarding, editing, and postproduction skills. |
| `visual/` | Visual design, image generation, prompting, motion, and 3D visualization skills. |

Use the narrowest subdirectory and skill file that matches the requested content output.
