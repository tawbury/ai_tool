# PM Skills Directory

이 디렉토리는 Project Manager(PM) 에이전트가 사용하는 기능 단위 스킬(`.skill.md`)들을 보관합니다.
프로젝트 기획, 일정 관리, 요구사항 정의, 이해관계자 조율 등의 역할을 지원합니다.

---

## 📂 파일 구성

| 파일명 | 스킬 역할 및 기능 |
|--------|-------------------|
| `pm_requirement_definition.skill.md` | 제품/기능 요구사항 정의(PRD) 작성 및 관리 |
| `product_lifecycle_management.skill.md` | 기획부터 배포, 유지보수까지 제품 수명 주기 전반 관리 |
| `pm_strategy_unified.skill.md` | 프로젝트 목표 설정 및 실행 전략 통합 수립 |
| `pm_analytics_unified.skill.md` | 성과 지표(Metrics) 분석 및 데이터 기반 의사결정 지원 |
| `product_risk_management.skill.md` | 프로젝트 리스크 식별 및 완화(Mitigation) 계획 수립 |
| `cross_functional_coordination.skill.md` | 타 직군(에이전트) 간 업무 조율 및 협업 관리 |
| `stakeholder_management.skill.md` | 이해관계자 피드백 수집 및 소통 관리 |
| `product_monetization.skill.md` | 비즈니스 모델 및 수익화 전략 기획 |
| `product_retention.skill.md` | 사용자 리텐션 강화를 위한 제품 개선 방안 분석 |
