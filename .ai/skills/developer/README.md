# Developer Skills Directory

이 디렉토리는 Developer 에이전트가 소프트웨어 엔지니어링, 아키텍처 설계, 코드 품질 관리 등을 수행하기 위해 사용하는 스킬(`.skill.md`)들을 보관합니다.

---

## 📂 주요 파일 구성

방대한 범위의 개발 역량을 세분화하여 다음과 같이 구성하고 있습니다:

| 파일명 | 스킬 역할 및 기능 |
|--------|-------------------|
| `dev_system_architecture.skill.md` | 전체 시스템 아키텍처 설계 및 검증 |
| `dev_backend_stack_unified.skill.md` / `dev_backend.skill.md` | 백엔드 프레임워크 기반 API/비즈니스 로직 구현 및 검증 |
| `dev_frontend_stack_unified.skill.md` / `dev_pwa.skill.md` | 프론트엔드 프레임워크 및 PWA(Progressive Web App) 구현 가이드 |
| `dev_database_design.skill.md` / `dev_database_optimization_unified.skill.md` | 데이터베이스 스키마 설계 및 쿼리/성능 최적화 |
| `dev_api_design.skill.md` | RESTful, GraphQL 등 API 인터페이스 명세화 |
| `dev_code_review.skill.md` / `code_quality_and_technical_debt_analysis.skill.md` | 소스코드 리뷰, 품질 지표 분석 및 기술 부채 관리 |
| `dev_testing.skill.md` | 단위, 통합, E2E 테스트 자동화 로직 작성 |
| `dev_devops_unified.skill.md` / `dev_cloud_architecture_unified.skill.md` | CI/CD 파이프라인 구성 및 클라우드 인프라 설계 |
| `dev_security.skill.md` | 보안 취약점 분석 및 안전한 코딩 가이드 |
| `dev_performance_optimization.skill.md` | 애플리케이션 성능 병목 탐색 및 개선 |
| `dev_chaos_engineering.skill.md` | 시스템 신뢰성 확보를 위한 장애 주입/테스트 |
| `dev_documentation.skill.md` | 기술 명세(Spec), 아키텍처 문서 생성 자동화 |
| `dev_deployment.skill.md` | 배포 프로세스 및 롤백 절차 자동화 |
