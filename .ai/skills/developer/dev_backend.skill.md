# Dev Backend Skill

<!-- BLOCK:CORE_LOGIC -->
## Core Logic
- Server-side component design/implementation/maintenance
- Robust/scalable/secure backend infrastructure creation
- Server-side development, database management, API creation, system architecture
<!-- END_BLOCK -->

<!-- BLOCK:INPUT_OUTPUT -->
## Input/Output
### Input
- Technical specification documents or development requirements
- System architecture definition
- Data model specifications

### Output
- Backend implementation plan
- API endpoint specifications
- Database schema
- Deployment configuration
<!-- END_BLOCK -->

<!-- BLOCK:EXECUTION_LOGIC -->
## Execution Logic
1. Analyze requirements & select technology stack
2. Design data models
3. Define API endpoint structures
4. Establish business logic implementation plan
5. Create security & performance optimization strategies
6. **Implement configuration management** (environment variables, config files)
7. **Establish path handling standards** (no absolute paths, cross-platform support)
8. Establish testing & deployment strategies
<!-- END_BLOCK -->

<!-- BLOCK:TECHNICAL_REQUIREMENTS -->
## Technical Requirements
- Server-side frameworks (Node.js, Python, Java)
- Database systems (SQL/NoSQL)
- API standards (REST, GraphQL)
- Authentication/authorization systems
- Monitoring/logging systems
<!-- END_BLOCK -->

<!-- BLOCK:CONSTRAINTS -->
## Constraints
### OUT Scope
- Frontend user interface development ❌
- User experience design ❌
- Customer support activities ❌
- Project management/planning ❌
- Marketing/sales activities ❌
- Budget/financial management ❌

### Technical Constraints
- Single responsibility principle compliance
- Scalability-considered design
- Security best practices application
- API consistency maintenance
- **Configuration externalization enforcement**
- **Path management best practices**
- **Environment-specific configuration separation**
<!-- END_BLOCK -->
