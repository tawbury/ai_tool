# Finance Skills Directory

이 디렉토리는 Finance 에이전트가 예산 기획, 비용 최적화, 리스크 검증, 그리고 금융/트레이딩 도메인의 업무를 수행할 때 사용하는 스킬(`.skill.md`)들을 보관합니다.

---

## 📂 파일 구성

| 파일명 | 스킬 역할 및 기능 |
|--------|-------------------|
| `strategic_financial_planning.skill.md` | 중장기 재무 계획 및 전략 수립 |
| `budget_management.skill.md` / `funding_management.skill.md` | 예산 할당, 비용 승인 및 자금 운영 기획 |
| `cost_optimization.skill.md` / `cash_flow_management.skill.md` | 리소스 비용 최적화 및 현금 흐름 분석 |
| `financial_analysis.skill.md` / `business_intelligence.skill.md` | 재무 제표 분석 및 BI 기반 수익성 모델링 |
| `market_trend_analysis.skill.md` | 거시 경제 및 시장 트렌드 조사, 영향도 분석 |
| `financial_risk_assessment.skill.md` / `compliance_management.skill.md` | 재무 리스크 식별 및 법률/규제 준수(Compliance) 검토 |
| `investment_portfolio_management.skill.md` | 투자 포트폴리오 관리 및 전략 점검 (트레이딩/금융 프로젝트 특화) |
| `forecasting_modeling.skill.md` | 수요 예측 및 재무 모델링 시뮬레이션 |
| `financial_reporting.skill.md` | 경영진용 재무 보고서 및 대시보드 데이터 산출 |
| `financial_system_optimization.skill.md` | 내부 재무/회계 파이프라인 및 시스템 최적화 |
