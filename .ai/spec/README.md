# Spec Directory

이 디렉토리는 MCP-cli 프레임워크 및 기반 시스템의 아키텍처, 운영 스펙 문서를 보관합니다.

---

## 📄 문서 구성 목록

| 문서명 | 주요 내용 |
|--------|-----------|
| `MCP_OS_Operational_Spec_v0_1.md` | Model Context Protocol 기반의 운영체제(OS) 수준 작동 원리, 아키텍처 사양, 에이전트 연동 스펙 등을 정의한 핵심 명세서 |

## 💡 주요 특징
- AI 에이전트의 프롬프트나 행동 지침이 아닌, **시스템 아키텍처 및 MCP 설계 원칙**을 기술합니다.
- 시스템 리팩토링이나 새로운 AI 클라이언트(Deepseek 등) 연동 시, 시스템이 어떤 스펙을 만족해야 하는지 기준이 되는 Reference 역할을 합니다.
