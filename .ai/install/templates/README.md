# Install Templates Directory

이 디렉토리는 `install` 프로세스 시 각 AI 클라이언트 폴더에 배포(복사)되는 초기화 파일 템플릿들을 보관합니다.

---

## 📂 파일 구성

| 파일명 | 배포 대상 및 목적 |
|--------|-------------------|
| `claude_template.md` | `.claude` 셋업 시 주입되는 룰/프롬프트 베이스 템플릿 |
| `copilot_template.md` | GitHub Copilot 인스트럭션 생성을 위한 베이스 템플릿 |
| `cursor_template.txt` | Cursor 에디터 룰 생성을 위한 초기 텍스트 템플릿 |
| `windsurf_template.txt` | Windsurf 에디터 설정 복사용 템플릿 |
