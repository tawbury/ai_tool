# 문서 디렉터리

이 디렉터리는 AIOS와 `.ai` 런타임의 계획, 감사, 구현 보고서, ADR, 로드맵, 사양 문서를 보관합니다.

## 현재 하위 디렉터리

| 경로 | 역할 |
|---|---|
| `00-pm/` | PM 관련 문서 공간 |
| `adr/` | Architecture Decision Record |
| `cc/` | Claude Code 관련 보조 문서 |
| `plan/` | 구현 전 계획 문서 |
| `reports/` | 감사, 구현, 검증 보고서 |
| `roadmap/` | 로드맵 문서 |
| `specs/` | 상세 사양 문서 |

## 규칙

- `docs/` 아래 문서는 한국어로 작성합니다.
- 계획과 보고서는 런타임 규칙이 아닙니다.
- 런타임에 적용할 규칙은 `.ai/rules/`의 가장 작은 관련 파일로 승격해야 합니다.
